## 개요
기존 퍼블릭 서브넷에서 EC2 배포시 보안성 취약문제를 개선한 배포과정
퍼블릭서브넷에는 ALB
프라이빗 서브넷에 각각 APP , DB 서버배치 (서로 다른 AZ  >> 고가용성 보장)
APP서버는 NAT instance(NAT gateway가 일반적인데 돈없어서) 로 퍼블릭접근
