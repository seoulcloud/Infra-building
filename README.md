## 아키텍쳐 개요
DB서버와 APP서버를 각각 퍼블릭 프라이빗 서브넷에 배치하고
APP서버에서만 DB서버 데이터에 접근하고 APP서버는 퍼블릭에서 접근가능한 아키텍쳐구성
